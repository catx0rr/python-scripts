# Weather.py

### Fetch weather data in terminal 

---
#### Usage:

```usage: weather.py [-h] [-l] [-c]```                                                                                                                                                                                                                                                                                                      
`optional arguments:                                                                                                                                                                                     
  -h, --help        show this help message and exit                                                                                                                                                     
  -l , --location   provide a country for a broader search.                                                                                                                                             
  -c , --city       provide a city for more accurate search.```
